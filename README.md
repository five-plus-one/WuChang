# 午唱管理程序

-----

## 缘由

在神秘的东方，[某学校书记](https://search.bilibili.com/all?keyword=%E9%9E%A0%E4%B9%9D%E5%85%B5)提出“每日一歌”之倡议，要求同学们在午觉睡醒之后全体起立唱歌，并将这一项任务纳入学校的常规考核。

同学们虽然有表不满，然而为了班级不被扣分，只能“欣然接受”这一提议。

在很多的班级，同学们都很踊跃，纷纷主动携带歌曲，在午休结束后立即起床播放歌曲；然而在我们班，同学们都睡的很死――显然，如果没有一个同学能够摆脱困意放歌，我们班级就会扣分；除非――我们可以做到自动播放。

于是，这样一个程序就诞生了。

现在，我们只需要在午休前运行这个程序，选择歌曲，程序就会先黑屏(防止屏幕太亮影响同学们睡觉)，然后在指定的时间亮屏并显示倒计时，最后在指定的时间播放歌曲――再也不用担心因为没睡醒不好放歌而被扣分啦！

## 使用

如果你不需要自定义，你可以直接前往[ releases 页](https://github.com/five-plus-one/WuChang/releases)下载相应的网页/程序直接使用，或访问<https://ntzx2215.five-plus-one.com/WuChang/>在线使用。

当然，你或许会不满意于里面的歌曲，认为这一点不符合你的品味，想要自定义。这时候，你需要前往[ releases 页](https://github.com/five-plus-one/WuChang/releases)对应的sb3文件自己编辑后打包。

虽然这个项目并没有使用 Scratch 原版不兼容的扩展，但是仍然建议您在 [Turbowarp](https://turbowarp.org/) 中编辑。将 sb3 文件导入到 Turbowarp 编辑器中，就可以自由添加你喜欢的歌了。
